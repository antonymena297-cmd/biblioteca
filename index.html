📖 Acerca del proyecto

Sistema completo para la gestión de bibliotecas que permite administrar libros, usuarios y préstamos de manera eficiente. Desarrollado con tecnologías modernas y buenas prácticas de programación.

⚙️ Tecnologías utilizadas

☕

Java 17

Lenguaje principal del backend

🍃

Spring Boot

Framework MVC y JPA

🐬

MySQL

Base de datos relacional

🎨

Thymeleaf

Motor de plantillas

💻

HTML5/CSS3

Interfaz de usuario

🐙

Git/GitHub

Control de versiones

🛠️ Funcionalidades

  • ✅ Agregar, editar y eliminar libros del catálogo
  • ✅ Buscar libros por título o autor
  • ✅ Verificar disponibilidad de libros
  • ✅ Registrar y eliminar usuarios
  • ✅ Realizar préstamos de libros disponibles
  • ✅ Registrar devoluciones
  • ✅ Historial de transacciones
  • ✅ Listar préstamos activos

📦 Cómo ejecutar localmente

# Clonar el repositorio
git clone https://github.com/antonymena297-cmd/biblioteca.git

# Entrar al directorio
cd biblioteca

# Configurar MySQL (crear base de datos)
mysql -u root -p -e "CREATE DATABASE biblioteca_db;"

# Ejecutar la aplicación
mvn spring-boot:run

# Abrir navegador en http://localhost:8080

📊 Estructura de la Base de Datos

📚 libros (id, titulo, autor, isbn, anio_publicacion, disponible)
👥 usuarios (id, nombre, apellido, cedula, email, telefono)
📅 prestamos (id, libro_id, usuario_id, fecha_prestamo, fecha_devolucion, devuelto)
